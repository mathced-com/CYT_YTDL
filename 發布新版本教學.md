# 發布新版本教學

這是一份給開發者 (您) 閱讀的專屬備忘錄。
當您修改了程式碼，想要發布新版本讓其他使用者能夠「自動下載更新」時，請依照以下步驟操作：

## 🚀 最簡單的方法：使用「一鍵發布腳本」
我們已經為您準備了 `一鍵發布新版本.bat`，這是最推薦且最不容易出錯的方法：

1. 雙擊執行 **`一鍵發布新版本.bat`**。
2. 依照黑視窗的提示，輸入「新的版本號」（例如：`1.0.1`）。
3. 接下來，腳本會自動幫您完成所有繁瑣的工作：
   - 幫您修改 `main.py` 裡面的版本號碼。
   - 幫您把程式打包成嶄新的 `CED_YTDL.exe` 執行檔。
   - 把最新的原始碼上傳、備份到 GitHub。
   - 自動幫您打開瀏覽器的 GitHub 發布網頁（並且把版本號都填好了！）。
4. **最後一步 (您需要手動做的唯一一件事)**：
   網頁打開後，您只需要把 `dist` 資料夾裡剛剛產生好的 `CED_YTDL.exe` 拖曳到網頁下方的上傳區塊，最後按下綠色的 **Publish release** 即可！

---

## 🛠️ 手動方法 (萬一腳本失效時的備用方案)
1. **修改版號**：打開 `main.py`，找到約第 16 行的 `APP_VERSION = "..."` 並修改為新版本號。
2. **打包程式**：點擊 `打包執行檔.bat` 產生出最新的 `.exe`。
3. **備份程式碼**：點擊 `上傳進度.bat` 把程式同步到雲端。
4. **發布到雲端**：
   - 到 [GitHub Releases](https://github.com/mathced-com/CED_YTDL/releases) 網頁。
   - 點擊 **Draft a new release**。
   - 填寫標籤 (Tag) 與標題（例如 `v1.0.1`）。
   - 將 `dist` 資料夾中的 `CED_YTDL.exe` 拖曳上傳。
   - 點擊 **Publish release**。
