# 🎵 CYT_YouTube 下載器：開發交接筆記 (v2.1.0)

### 1. 目前的開發目標
建立一個整合 YouTube 下載、MP3 裁剪與 MP3 跨首歌融合合併的桌面工具。目前的 v2.1.0 版本已達到功能穩定，重點在於優化音訊試聽的流暢度與自動更新系統的穩定性。

### 2. 核心程式碼快照 (關鍵邏輯)
```python
# 雙播放器無縫銜接與融合預覽邏輯
def _do_update(self):
    p_active = self.players[self.active_player_idx]
    time_left = self.staged_durations[self._current_song_idx] - p_active.get_position()
    
    # 預加載下一首 (提前 3 秒)
    if has_next and time_left <= 3000 and not self._next_song_triggered:
        other_p = self.players[1 - self.active_player_idx]
        other_p.open(next_file)
        if self.fade_var.get(): other_p.play(); other_p.set_volume(0)
        self._next_song_triggered = True

    # 執行音量漸變 (MCI setaudio volume)
    if self.fade_var.get() and time_left <= fade_ms:
        p_active.set_volume(int(fade_ratio * 1000))
        other_p.set_volume(int((1-fade_ratio) * 1000))

    # 瞬間切換
    if time_left <= 120:
        self.active_player_idx = 1 - self.active_player_idx
        if not self.fade_var.get(): self.players[self.active_player_idx].play()
```
