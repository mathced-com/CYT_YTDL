# 發布新版本教學

這是一份給開發者 (您) 閱讀的專屬備忘錄。
當您修改了程式碼，想要發布新版本讓其他使用者能夠「自動下載更新」時，請依照以下步驟操作：

## 🚀 最簡單的方法：使用「一鍵發布腳本」
我們已經為您準備了 `一鍵發布新版本.bat`，這是最推薦且最不容易出錯的方法：

1. 雙擊執行 **`一鍵發布新版本.bat`**。
2. 依照黑視窗的提示，輸入「新的版本號」（例如：`1.0.1`）。
3. 接下來，腳本會自動幫您完成所有繁瑣的工作：
   - 幫您修改 `main.py` 裡面的版本號碼。
   - 幫您把程式打包成嶄新的 `CYT_YTDL.exe` 執行檔。
   - 把最新的原始碼上傳、備份到 GitHub。
   - **(最新功能) 自動將 `CYT_YTDL.exe` 上傳到 GitHub Releases 中！**
4. **最後確認**：
   發布完成後，程式會自動幫您打開瀏覽器的 GitHub 發布網頁，這只是為了讓您「確認」檔案已經成功上傳，您不需要做任何額外操作。

---

## 🛠️ 手動方法 (萬一腳本失效時的備用方案)
1. **修改版號**：打開 `main.py`，找到約第 16 行的 `APP_VERSION = "..."` 並修改為新版本號。
2. **打包程式**：點擊 `打包執行檔.bat` 產生出最新的 `.exe`。
3. **備份程式碼**：點擊 `上傳進度.bat` 把程式同步到雲端。
4. **發布到雲端**：
   - 到 [GitHub Releases](https://github.com/mathced-com/CYT_YTDL/releases) 網頁。
   - 點擊 **Draft a new release**。
   - 填寫標籤 (Tag) 與標題（例如 `v1.0.1`）。
   - 將 `dist` 資料夾中的 `CYT_YTDL.exe` 拖曳上傳。
   - 點擊 **Publish release**。

---

## 📢 附錄：分享軟體給他人時的「防毒軟體免責聲明」範本

因為本程式未購買昂貴的微軟數位簽章憑證，當您將 `CYT_YTDL.exe` 傳給朋友或同事時，他們的電腦很可能會跳出藍色的 Windows 阻擋畫面。

您可以**直接複製以下這段文字**，在傳送軟體時一併貼給他們看：

> **【關於首次開啟的安全性提示】**
> 
> 哈囉！這支「CYT_YouTube 下載器」是我個人獨立開發的實用小工具。
> 
> 由於這支程式是免費分享，沒有花費每年數千元向微軟購買「數位簽章憑證」，因此當您第一次點擊 `CYT_YTDL.exe` 執行時，Windows 系統通常會跳出藍色的「**Windows 已保護您的電腦**」警告視窗。
> 
> 這只是微軟針對「未購買憑證的未知發行者」的標準防護機制，**請放心，程式保證絕對無毒！**
> 
> **👉 解除阻擋的方法非常簡單：**
> 1. 在藍色警告視窗中，點擊左上角白色的字體 **「其他資訊」**。
> 2. 接著點擊右下角出現的 **「仍要執行」** 按鈕。
> 
> 這樣就能順利開啟並永久加入白名單囉！祝您使用愉快！
