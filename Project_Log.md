## 🏗️ 核心架構與技術棧
- **開發語言**：Python 3
- **GUI 介面**：`tkinter` + `ttk` (內建，無需額外安裝)
- **核心下載引擎**：`yt-dlp` (負責解析網址與下載影音串流)
- **多媒體處理**：`FFmpeg` (`ffmpeg.exe` 與 `ffprobe.exe`，負責影音合併與 MP3 轉檔)
- **影像處理**：`Pillow` (PIL) (負責在介面中顯示 YouTube 影片縮圖)
- **非同步處理**：使用 `threading` 模組，確保下載過程中介面（GUI）不會卡頓當機。

## 📁 主要檔案結構說明
- **`main.py`**：核心程式碼。包含 `YouTubeDownloaderGUI` 類別，負責所有視窗排版、使用者互動、網址解析與呼叫下載進程。
- **`setup.bat`**：環境初始化腳本。會自動更新 pip、安裝必要的 Python 套件 (`yt-dlp`, `Pillow`)，並呼叫 `install_ffmpeg.py`。
- **`install_ffmpeg.py`**：自動檢查或下載安裝 FFmpeg 的 Python 腳本。
- **`啟動下載器.bat`**：讓使用者雙擊即可直接啟動 `main.py` 的便利腳本，不需透過終端機。
- **`上傳進度.bat` / `下載進度.bat`**：用於 GitHub 雲端同步的一鍵腳本，方便跨電腦開發。
- **`.gitignore`**：Git 排除清單，防止將巨大的 `ffmpeg.exe` 或下載的影片同步到雲端。

## 🚀 主要功能清單
1. 支援單一影片解析與下載。
2. 支援 YouTube 播放清單解析（可勾選特定集數）。
3. 支援 MP4（影片）多種畫質選擇（最高、1080p、720p 等）。
4. 支援 MP3（音訊）多種音質選擇（320k、192k、128k）。
5. 具備即時進度條、下載速度與剩餘時間顯示。
6. 具備「暫停 / 繼續」與「取消」下載功能。
